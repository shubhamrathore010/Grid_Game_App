## Getting Started

### Prerequisites

- Node.js (v14+ recommended)  
- npm or yarn  

### Installation

```bash
git clone https://github.com/shubhamrathore010/Grid_Game_App.git
cd Grid_Game_App
npm install
```

### Running Locally

```bash
npm run dev
```
### Production Build

```bash
npm run build
```
